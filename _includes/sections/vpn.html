Mullvad {% include badge.html color="info" text="歐元 €60/y" %}

Mullvad.net 速度快、價格合理的 VPN，同時認真看待安全與透明課題。自 2009開始營運，Mullvad 主要位於 瑞典，未提供免費試用

{% include badge.html color="success" text="35 Countries" %}

Mullvad 在 35 個國家安置伺服器。 地理位置也應放入考量，最好能挑選距離最近的 VPN 伺服器，可因最短路徑而減少流量延遲。

基於安全考量，我們認為最好使用專用伺服器而非廉價共用的伺服器方案(例如虛擬私有伺服器)，以確保私鑰安全。

Mullvad 的 VPN 客戶端軟體曾由 Cure53 與 Assured AB 稭核，提交滲透測試報告，公佈在 cure53.de。 安全檢查的研究成員包括：

...Cure53 與 Assured AB 對於安全稭核結果感到滿意，對他們的軟體留下了良好印象。Mullvad VPN 自家團隊在資安用心的專注投入，測試員深信此專案朝著正確的軌道前進。

{% include badge.html color="success" text="軟體公開代碼" %}

Mullvad 將其電腦與手機客戶端軟體的源代碼公開在 GitHub.

{% include badge.html color="success" text="接受比特幤付款" %}

Mullvad 可採 信用卡/簽帳卡、比特幣、Bitcoin Cash以及現金或當地流通貨幣等不記名付款方式，也可接受 Swish 與銀行電滙。

{% include badge.html color="success" text="WireGuard 支援" %}

除標準的 OpenVPN 連線， Mullvad 還支援 WireGuard。WireGuard 是試驗階段的網路協定，理論上會更安全與更可靠。不過目前它仍未大量使用。

{% include badge.html color="success" text="IPv6 支援" %}

Mullvad 支援 IPv6 網路協定，讓用戶可訪問架設在 IPv6的服務。不少供應商會阻斷 IPv6 連線。

{% include badge.html color="success" text="遠處端口轉發 Remote Port Forwarding" %}

Mullvad 提供遠端端口轉發功能，請見Port forwarding with Mullvad VPN。

{% include badge.html color="success" text="手機端軟體" %}

Mullvad 提供手機應用，可在App Store 和 Google Play 下載客戶端軟體以簡化 WireGuard 連線設定。安卓版應用也可至F-Droid下載，以確保該應用符合 可複製建置原則。

{% include badge.html color="info" text="額外功能" %}

The Mullvad VPN 客戶端內鍵 killswitch 以阻斷 VPN 以外的網路連線，也能在開機時即自動載入 VPN。 Mullvad 網站可以透由 Tor xcln5hkbriyklr6n.onion 訪問。

{% include badge.html color="success" text="手機應用" %}

IVPN 標準版美元 $60/年 昇級版 美元 $100/年

IVPN.net是另一家不錯的 VPN 供應商，成立於 2009 年，註冊所在地是 直布羅陀。

{% include badge.html color="success" text="32 Countries" %}

截至目前統計，IVPN 在全球 32個國家設置伺服器，請挑選距離最近的伺服器，可因最短路徑而減少流量延遲。

基於安全考量，我們認為最好使用專用伺服器而非廉價共用的伺服器方案(例如虛擬私有伺服器)，以確保私鑰安全。

{% include badge.html color="success" text="獨立稭核" %}

IVPN 曾委託 Cure53 進行 no-logging 稭核，其證明了 IVPN's 無保留用戶記錄的聲明， 2020 年元月IVPN 也執行了has also completed a 完整的透過報告，表示未來會定期發佈年報。

{% include badge.html color="success" text="開源的客戶端軟體" %}

自從 2020 年 2月IVPN 應用軟體已公開其源代碼，請參考其GitHub 組織帳號。

{% include badge.html color="success" text="接受比特幤付款" %}

除常見的信用卡 PayPal, IVPN 可收 比特幣 與 當地貨幣現金等匿名付款方式。

{% include badge.html color="success" text="WireGuard 支援" %}

IVPN 支援標準的 OpenVPN 與 WireGuard 連線，後者是仍在試驗的協定，理論上會更安全可靠。目前不建議對 WireGuard 進行大量使用。

{% include badge.html color="success" text="遠處端口轉發 Remote Port Forwarding" %}

付費昇級的用戶可享遠端遠處端口轉發功能，透過客戶端軟體開啟。只有在使用 OpenVPN 且不連接位在美國的伺服器時，才可進行端口轉發功能。

{% include badge.html color="success" text="手機應用" %}

除標準的 OpenVPN 設定外， IVPN 也有手機專用的 iOS 與 Android 應用軟體，便於用戶連接到他們的伺服器。安卓版應用也可至F-Droid下載，以確保該應用符合 可複製建置原則。

{% include badge.html color="info" text="額外功能" %}

IVPN 客戶端軟體內建 killswitch 以阻斷 VPN 之外的網際連線，它會在開機時自動啟動， IVPN 有"AntiTracker" 功能，可阻檔廣告或追蹤器。

注意：使用 VPN 無法匿名，但某些狀況下提供用戶更佳的隱私保護。 VPN 不是用來從事違法活動的工具，不要依賴「不留活動記錄 "no log"」政策。
