值得一提

  • FramaDrop - 可寄放不限大小的檔案達24小時，透過用戶瀏覽器執行端對端加密，由LuFi開發。 {% include badge.html color="warning" icon="fas fa-exclamation-triangle" link="https://framasoft.org/en/cgu/" tooltip="FramaDrop logs IP addresses and fingerprints the browser for an unclear amount of time." %}
  • croc - 好用且能安全進行電腦資料傳送的工具。
  • FreedomBox - 設計用於自架的經濟型家用伺服器，可運行自由軟體和提供越來越多的服務，例如日曆、Jabber server, wiki 或 VPN 等。
